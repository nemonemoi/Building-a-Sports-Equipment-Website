Header()

설명

이 메소드를 헤더를 만들때 사용합니다. AddPage() 에 의해 자동으로 호출되므로 프로그램내에서 일부러 호출할 필요는 없습니다. FPDF 내에는 구현되지 않은채 존재하므로 별도의 처리를 원한다면 서브클래스를 구현해서 메소드를 오버라이드 해야합니다.

예제

class PDF extends FPDF
{
function Header()
{
    //Select Arial bold 15
    $this->SetFont('Arial','B',15);
    //Move to the right
    $this->Cell(80);
    //Framed title
    $this->Cell(30,10,'Title',1,0,'C');
    //Line break
    $this->Ln(20);
}
}
