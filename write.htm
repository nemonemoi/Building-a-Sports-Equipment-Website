Write

Write(float h, string txt [, mixed link])

설명

이 메소드는 현재 위치에 문자열을 출력합니다. 출력도중 오른쪽 가장자리에 도달하면 (혹은 \n 문자를 만나면) 줄을 바꾸고 다음 줄의 왼쪽 가장자리부터 텍스트를 계속 출력합니다. 메소드를 종료하면 현재 위치는 끝 문자의 바로 옆이 됩니다.
또한 텍스트에 링크를 출력하는 것도 가능합니다.

파라미터

h
줄 높이
txt
출력할 문자열
link
URL이나 AddLink() 실행후 얻은 고유값

예제

//보통의 글꼴로 시작
$pdf->SetFont('Arial','',14);
$pdf->Write(5,'Visit ');
//밑줄이 쳐진 파란색 링크문자열을 출력
$pdf->SetTextColor(0,0,255);
$pdf->SetFont('','U');
$pdf->Write(5,'www.fpdf.org','http://www.fpdf.org');

참고

SetFont(), SetTextColor(), AddLink(), MultiCell(), SetAutoPageBreak().
